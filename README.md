# neptune-ai-tutorial

A small tutorial for neptune.ai

### Getting Started
1. Clone the repo
```
git clone https://github.com/RektPunk/neptune-ai-tutorial.git
```

2. Install requirements
```
pip install -r requirements.txt
```

3. Create file named `.env` and add neptune api token.

```python
NEPTUNE_API_TOKEN = "***"
```
